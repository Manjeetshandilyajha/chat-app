# 💬 Real-Time Chat Application

A modern real-time chat application built using the MERN Stack and Socket.IO. It enables users to communicate instantly through a fast, secure, and responsive messaging platform.

## 🚀 Features

- 💬 Real-Time Messaging
- 👤 User Authentication (Login & Register)
- 🟢 Online/Offline Status
- 📩 One-to-One Chat
- 😊 Emoji Support
- 🔔 Instant Notifications
- 📱 Fully Responsive Design
- 🔒 Secure Authentication with JWT
- ⚡ Fast and User-Friendly Interface

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Real-Time Communication
- Socket.IO

### Authentication
- JWT
- bcrypt.js

---

## 📦 Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/chat-app.git
```

### Move to Project Folder

```bash
cd chat-app
```

### Install Dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd server
npm install
```

---

## ⚙️ Environment Variables

Create a `.env` file inside the server folder.

```env
PORT=5000

MONGO_URI=Your_MongoDB_URL

JWT_SECRET=Your_JWT_SECRET

CLIENT_URL=http://localhost:5173
```

---

## ▶️ Run the Application

Backend

```bash
cd server
npm run dev
```

Frontend

```bash
cd client
npm run dev
```
